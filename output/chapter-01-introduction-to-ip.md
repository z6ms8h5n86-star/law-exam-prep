# 第一章 知识产权导论 (Introduction to Intellectual Property)

---

## 1. 知识产权：概念困境与课程定位

### （1）核心概念与定义

**【★重点】** 知识产权（Intellectual Property, IP）不存在统一的全球定义（There is no universal definition of IP）。这是一个根植于学科底层的事实——IP 主要通过举例（enumeration by examples）来描述，而非通过属加种差式的经典定义方式。

知识产权法（IP Law）的规范对象可概括为：

> IP Law regulates the human creation, use and exploitation of mental or creative labour.

——知识产权法调整人类对智力或创造性劳动成果的创造、使用和利用行为。

另一种更诚实的学术叙述（a more honest account）则指出：

> IP starts "from the premise or foundation that ideas are free as the air — a common resource."

——知识产权制度的起点是"思想如空气般自由——属于公共资源"这一前提。这一表述揭示了 IP 制度的内在张力：它是在公共领域（the commons / public domain）之上人为创设的排他性权利。

**中国官方定位（2021）**：中共中央、国务院《知识产权强国建设纲要（2021－2035年）》明确指出——

> 知识产权是"国家发展战略性资源和国际竞争力核心要素"
> IP is a core of global economy and global governance.

**定义困境的三层原因**：

| 层次 | 困境 | 说明 |
|------|------|------|
| 对象层面 | IP 客体种类繁多且不断扩张 | 从传统的专利/商标/版权扩展到集成电路布图、植物新品种、数据等，难以用一个上位概念统摄 |
| 性质层面 | 无形财产与有形财产规则的不适配 | IP 参照有形财产权（tangible property）规则构建，但无形性导致直接套用会产生垄断过度的问题 |
| 功能层面 | 激励创新 vs 阻碍流通的根本矛盾 | IP 究竟是鼓励创新的流通还是鼓励创新的出现？是否会形成对知识的垄断（monopoly on knowledge）？ |

### （2）制度原理与体系

#### 2.1 举例式定义法 —— 为什么 IP 难以下定义

**【★重点】** IP 的界定高度依赖举例（enumeration），三种主要国际/国内法律文件对 IP 范围的列举各不相同：

```
┌──────────────────────────────────────────────────────────────────┐
│         知识产权客体范围的三重列举体系                            │
├────────────────┬─────────────────┬──────────────────────────────┤
│  WIPO 公约     │   TRIPs 协定    │   中国《民法典》第123条       │
│  (1967)        │   (1994)        │   (2021)                     │
│  8 大类        │   7 大类        │   8 类客体                   │
├────────────────┼─────────────────┼──────────────────────────────┤
│ 1. 文学艺术    │ 1. 版权与邻接权 │ 1. 作品                      │
│ 2. 表演        │ 2. 商标权       │ 2. 发明、实用新型、外观设计  │
│ 3. 发明        │ 3. 地理标志权   │ 3. 商标                      │
│ 4. 科学发现    │ 4. 工业品外观   │ 4. 地理标志                  │
│ 5. 工业品外观  │    设计权       │ 5. 商业秘密                  │
│ 6. 商标商号    │ 5. 专利权       │ 6. 集成电路布图设计          │
│ 7. 反不正当    │ 6. 集成电路布图 │ 7. 植物新品种                │
│    竞争        │    设计权       │ 8. 法律规定的其他客体        │
│ 8. 兜底条款    │ 7. 未披露信息   │                              │
│                │    专有权       │                              │
└────────────────┴─────────────────┴──────────────────────────────┘
```

#### 2.2 三类列举的关键差异

**【★重点】WIPO 公约 vs TRIPs 协定的差异（考试高频对比点）**：

| 比较维度 | WIPO 公约（1967） | TRIPs 协定（1994） |
|----------|-------------------|---------------------|
| 表述方式 | 一般性、包容性表述 | 有明显美国特色（American flavor）的表述 |
| 地理标志 | 未单独列出 | **特别涉及地理标志权**（geographical indications） |
| 外观设计 | 笼统列在"工业品外观设计" | 外观设计与专利**分列**为两项独立权利 |
| 特色内容 | 包含"科学发现"（有争议） | 包含**集成电路布图设计权**（layout-designs of integrated circuits），体现美国半导体产业利益 |
| 兜底条款 | 有明确的兜底条款 | 通过"未披露过的信息专有权"（undisclosed information）实质上实现对商业秘密的保护 |
| 法律性质 | 多边条约，协调性为主 | WTO 框架下有强制约束力（"带牙齿的"） |

**中国《民法典》第123条的独特性**：
- 将**植物新品种**（plant varieties）单独列为 IP 客体——这是中国特色的体现
- "法律规定的其他客体"——国内学者关注的核心问题是**数据保护**（data protection）能否纳入
- 采用"专有权利"（exclusive rights）的表述，与 TRIPs 的财产权话语保持距离

#### 2.3 知识产权保护期限对比

**【★重点】** 三类主要 IP 的保护期限存在显著差异，反映了不同政策考量：

| IP 类型 | 保护期限 | 政策理由 |
|---------|---------|----------|
| **专利权** (Patents) | 15-20 年 | 以临时垄断换取技术公开（quid pro quo），期限较短以促进技术迭代 |
| **著作权** (Copyrights) | 70-170 年（自然人死后 70 年 / 公司持有 70-170 年） | 激励文化创作，保护期较长以保障作者及其继承人利益。公司持有期限更长反映了产业利益驱动 |
| **商标权** (Trademarks) | **无限期保护** (Indefinite protection) | 商标是识别来源的标志，只要持续使用并续展，不存在到期问题。包括地理标志（Geographical Indications，如阳山水蜜桃） |

### （3）案例与适用

**参考案例：开源运动（Open Source Movement）的知识产权悖论**

开源软件采用"著佐权"（copyleft）许可协议（如 GPL）——使用开源代码者必须同样以开源方式发布其衍生作品。这一模式提出了一个根本性追问：

> 如果开源（脱离 IP 排他性保护）能够驱动如此广泛的创新（Linux、Android 等），那么 IP 是否真的是创新的核心驱动力？开源是否才是创新之本？

这一问题直指知识产权正当性理论的核心争议，考试中可能以论述题形式出现。

**参考案例：美国版权保护的历史反转**

19 世纪的美国是一个典型的"知识盗版国"（pirate nation）——作为一个移民社会，美国仅对美国作者提供版权保护，大量盗版欧洲（尤其是英国）文学作品。欧洲曾派代表团专程游说美国加强版权保护，未果。今天美国成为全球最大的文化输出国后，立场完全反转，成为全球最积极推动高标准 IP 保护的国家。

> 这一历史案例说明：**知识产权制度有深刻的政治文化层面的投射**，一国的 IP 立场与其在全球产业链中的位置密切相关。

### （4）考试提示

- **名词解释**："知识产权"一词的定义困境是经典考题——必须点出"无统一定义 + 举例式方法 + 无形性"
- **简答**：WIPO vs TRIPs 的差异，至少答出 3 点（表述方式、地理标志、集成电路布图 + 外观设计分列）
- **论述**：IP 保护期限差异背后的政策逻辑，尤其注意商标无限期保护的原因（来源识别功能）
- 开源案例和美国版权反转案例是论述题的高质量素材

## 2. 知识产权的本质属性：四组根本性张力

### （1）核心概念与定义

**【★重点】** 知识产权的本质（the nature of IP）不是单一维度的，而是体现在四组对立统一的张力之中。这四组张力构成了理解整个 IP 制度的基础理论框架。

### （2）制度原理与体系

#### 2.1 第一组张力：无形（Intangible）vs 有形（Tangible）

```
┌─────────────────────────────────────────────────────────────────┐
│              知识产权无形性 vs 有形财产权                        │
├──────────────────────┬──────────────────────────────────────────┤
│      有形财产        │           知识产权（无形）                │
├──────────────────────┼──────────────────────────────────────────┤
│ 占有具有排他性       │ 可被多人同时使用而不减损（non-rivalrous） │
│ 权利边界清晰         │ 权利边界模糊，需法律人为界定              │
│ 通过占有即可保护     │ 必须通过法律赋予排他权才能保护            │
│ 不存在期限问题       │ 一般有法定期限（商标为例外）              │
│ 地域性较弱           │ 严格的地域性（territoriality）            │
└──────────────────────┴──────────────────────────────────────────┘
```

**核心问题**：知识产权本质上是无形的（intangible），但法律规制时参照了有形财产权（tangible property）的框架。这种"类比规制"（analogy-based regulation）带来两个后果：
1. **正效果**：借助成熟的财产权理论快速建立起 IP 保护体系
2. **负效果**：无形财产不具排他占有属性，若完全按有形财产逻辑会**造成大量垄断**（excessive monopolies）

**学术争议**：知识产权是否应当脱离财产权范式，建立独立的"无形权利"理论体系？

- **支持脱离派**：无形性与有形性存在本质区别，强行套用导致制度扭曲
- **维持现状派**：财产权范式虽有缺陷但具有操作性优势，渐进改良优于推倒重来

#### 2.2 第二组张力：自然权利（Natural Rights）vs 国家授予（State-Granted）

| 维度 | 自然权利说 | 国家授予说 |
|------|-----------|-----------|
| 权利来源 | 创作者基于创造性劳动**天然**获得 | 国家通过法律**授予**才产生 |
| 理论基础 | 洛克劳动财产论（Locke's labor theory） | 功利主义（utilitarianism） |
| 保护逻辑 | 保护创作者的精神与物质利益 | 以保护换取创新，促进公共利益 |
| 政策含义 | 倾向于强保护、长保护期 | 倾向于平衡保护，注重公共领域 |
| 代表人物 | 欧洲大陆传统（尤其法国"作者权"） | 英美传统（尤其美国宪法"促进科学与实用技艺进步"） |

**老师的立场**（来自课程笔记）：自然权利说的论证面临根本困难——如果创作即产生自然权利，为什么 IP 有期限而其他自然权利（如生命权）无期限？为什么需要国家法律来"确认"这种"自然"权利？这恰恰说明 IP 并非自然权利，而是一种**政策工具**（policy instrument）。

#### 2.3 第三组张力：私有财产（Private Property）vs 垄断（Monopoly）

**【★重点】** 这是 IP 领域最核心的学术争议之一。

- **"垄断"（monopoly）标签**：西方主流话语普遍将 IP 视为垄断。Monopoly 带有负面价值判断，暗示对竞争的不当限制。
- **"排他性权利"（exclusive rights）标签**：中性的、技术性的表述。邵科教授在课程中明确指出 exclusive rights 是**核心描述词**（core descriptor）。
- **为什么表述之争很重要？** 概念的选择影响制度设计的方向——如果你认为 IP 是"垄断"，你就会倾向于限制它；如果你认为 IP 是"排他性权利"，你就会倾向于保护它。这是话语权的争夺。

```
垄断 (Monopoly) ──── 价值判断 ──── 负面：限制竞争，应受约束
                                       │
        排他性权利 (Exclusive Rights) ── 技术描述 ── 中性：权利边界，依法界定
```

#### 2.4 第四组张力：排他性权利 vs 公有领域（The Public Domain / The Commons）

**【★重点】** IP 制度是在公有领域之上划定的排他性"岛屿"。

**公有领域排除原则**：
- 发现（discoveries）——如科学公式、自然规律——不予以 IP 保护，因为它们属于公有领域
- 抽象思想（abstract ideas）——不予以保护
- 保护**期限**届满后——作品/发明回归公有领域（进入 public domain）

**核心追问**：公有领域的范围是不是太小了？

- **批判方**：IP 保护范围不断扩张（保护期延长、客体扩展），公有领域被持续侵蚀。计算机领域的**开源运动**（Open Source）恰恰证明脱离 IP 排他逻辑可以驱动强大的创新。
- **辩护方**：IP 制度从未否定公有领域——保护期限制本身就是在保障公有领域的最终回归。IP 是在公有领域的基础上**临时**"圈地"以激励创新。

**开源运动的挑战**：计算机领域采用开源协议（如 GPL）——"用开源，自己也必须开源"的传染机制——是否说明 IP 不是创新的核心，开源才是？这一问题没有标准答案，但必须理解两种逻辑的对立。

### （3）案例与适用

**思想实验：如果公式可以申请专利**

爱因斯坦发现 E=mc² ——这是对自然界规律的**发现**（discovery），不是**发明**（invention）。如果允许对科学公式授予专利：
- 后果 1：每次使用该公式都需要向权利人付费——基础科学研究将停滞
- 后果 2：后续所有基于该公式的技术创新都被封锁
- 结论：公有领域的保有是后续创新的前提条件。这正是为什么"科学发现"（scientific discoveries）在 WIPO 公约中被列为 IP 种类，但实际上绝大多数国家不保护科学发现本身（中国是否有"科学发现权"存有争议）。

### （4）考试提示

- **论述题必考方向**："知识产权的本质属性"——需全面展开四组张力，每组张力至少给出正反两方观点
- **概念辨析**：Monopoly vs Exclusive Rights 的表述差异及其政策含义
- **简答题**：公有领域（public domain）的范围与功能
- 判标题中区分"发现"（不受保护）与"发明"（可受保护）是常见考点

## 3. 传统分类体系与国际/国内法上的 IP 种类

### （1）核心概念与定义

知识产权在传统大陆法系中被划分为两大基本类型：

```
传统知识产权分类
│
├── 工业产权 (Industrial Property)
│   ├── 专利权 (Patent Right)
│   └── 商标权 (Trademark Right)
│
└── 著作权/版权 (Copyright / Author's Right)
```

**工业产权**主要保护产业领域的创造性成果和识别性标记。
**著作权**保护文学、艺术和科学领域的创造性表达。

### （2）制度原理与体系

#### 3.1 《建立世界知识产权组织公约》（WIPO Convention, 1967）—— 8 大类

WIPO 公约第 2 条对 IP 权利种类作了最广泛的列举：

| 序号 | WIPO 公约 — 8 大类 | 说明 |
|------|-------------------|------|
| 1 | 文学、艺术和科学作品 | 对应著作权/版权的核心领域 |
| 2 | 表演艺术家的表演、录音和广播 | 邻接权（neighboring rights / related rights） |
| 3 | 在人类一切活动领域内的发明 | 发明专利 |
| 4 | **科学发现** | 争议项——大多数国家不独立保护科学发现，中国是否有科学发现权存有争议 |
| 5 | 工业品外观设计 | 多数国家单独立法保护 |
| 6 | 商标、服务标记、商号及商业名称 | 识别性标记 |
| 7 | 反不正当竞争保护 | 主要指商业秘密（trade secrets）的保护 |
| 8 | 以及在工业、科学、文学或艺术领域内其他一切来自知识活动的权利 | 兜底条款（catch-all provision） |

#### 3.2 TRIPs 协定（1994）—— 7 大类

TRIPs 协定（Agreement on Trade-Related Aspects of Intellectual Property Rights）第 1 条第 2 款规定：

| 序号 | TRIPs 协定 — 7 大类 | 与 WIPO 的关键差异 |
|------|---------------------|-------------------|
| 1 | 版权与邻接权 | 合并了 WIPO 的前两项 |
| 2 | 商标权 | 基本一致 |
| 3 | **地理标志权** | WIPO 未单独列出的新增项 |
| 4 | 工业品外观设计权 | 与专利**分列**（WIPO 中发明和外观均为笼统列举） |
| 5 | 专利权 | 对应 WIPO 第 3 项 |
| 6 | **集成电路布图设计权** | TRIPs 特色，体现美国半导体产业利益 |
| 7 | 未披露过的信息专有权 | 涵盖商业秘密（trade secrets / undisclosed information） |

#### 3.3 中国《民法典》第 123 条 —— 8 类客体

**【★重点】** 《民法典》第 123 条是中国 IP 保护体系的最高位阶民事法律规范：

> 知识产权是权利人依法就下列客体享有的**专有权利**：……

| 序号 | 民法典第123条客体 | 特色说明 |
|------|------------------|----------|
| (一) | 作品 | 著作权法保护对象 |
| (二) | 发明、实用新型、外观设计 | 三种专利类型统一列举 |
| (三) | 商标 | 商标法保护对象 |
| (四) | 地理标志 | 与 TRIPs 对齐 |
| (五) | 商业秘密 | 与 TRIPs"未披露信息"对应 |
| (六) | 集成电路布图设计 | 与 TRIPs 对齐 |
| (七) | **植物新品种** | 中国特色——中国是农业大国，植物新品种保护具有重要战略意义 |
| (八) | 法律规定的其他客体 | 兜底条款——**国内学者核心关注：数据保护** |

**【★重点】民法典第123条的结构分析**：

1. **权利主体**："权利人"——未限定为自然人/法人，开放性表述
2. **权利来源**："依法"——强调 IP 是法定权利，非自然权利。没有法律规定则无 IP
3. **权利性质**："专有权利"（exclusive rights）——中性表述，避免"垄断"的价值标签
4. **保护客体**：8 类 + 兜底——将国际义务国内法化，且增加了植物新品种
5. **兜底条款的功能**：为未来新类型 IP 客体（尤其是数据）预留立法空间

#### 3.4 三重体系的全景对比

```
                        知识产权客体范围
                              │
        ┌─────────────────────┼─────────────────────┐
        │                     │                     │
   WIPO 公约              TRIPs 协定          中国民法典123条
   (8大类,1967)           (7大类,1994)         (8类客体,2021)
        │                     │                     │
   · 最广泛列举          · 美国特色明显        · 综合吸收WIPO+TRIPs
   · 含科学发现(争议)    · 地理标志独立        · 增加植物新品种
   · 有兜底条款          · 集成电路布图        · 兜底→数据保护展望
   · 协调性多边条约      · 含强制执行力        · 最高民事法律位阶
```

### （3）案例与适用

**中国是否有"科学发现权"？**

WIPO 公约将"科学发现"列为 IP 种类，但中国国内法并未设立独立的"科学发现权"。《民法典》第 123 条的前 7 项具体列举中不包括科学发现。科学发现在中国主要通过以下方式获得间接保护：
- 如果将科学发现以"作品"形式发表——受著作权保护（保护的是表达，不是发现本身）
- 如果基于科学发现做出了"发明"——可申请专利保护（保护的是技术方案，不是发现本身）

这体现了 IP 法的基本原则：**发现属于公有领域，不被任何人独占**。

### （4）考试提示

- **简答题高频**："比较 WIPO 公约与 TRIPs 协定对知识产权种类的规定"——建议用表格形式作答，至少指出 3-4 点差异
- **名词解释**：TRIPs 协定（全称必须写出：Agreement on Trade-Related Aspects of Intellectual Property Rights）
- **民法典第 123 条**是论述题的核心法条——结构分析（主体+来源+性质+客体+兜底）要能完整展开
- **植物新品种**——中国特色的 IP 客体，为什么是农业大国的战略选择？可联系《知识产权强国建设纲要》

## 4. 知识产权国际公约体系：从巴黎到 TRIPs 的演进

### （1）核心概念与定义

知识产权的国际保护始于 19 世纪末，经历了从协调性双边/多边条约到具有强制执行力的全球性体制的演进。理解这一演进，是理解当代 IP 全球治理格局的前提。

### （2）制度原理与体系

#### 4.1 演进全景图

```
1883                    1886                 1967/1970            1994
 │                       │                      │                   │
 ▼                       ▼                      ▼                   ▼
巴黎公约              伯尔尼公约              WIPO               WTO/TRIPs
(Paris Convention)    (Berne Convention)    (世界知识产权组织)    (与贸易有关的知识产权协定)
 │                       │                      │                   │
 ▼                       ▼                      ▼                   ▼
第一个重要IPR条约    版权全球普及的起点    两联盟国际局合并      "带牙齿的国际经济组织"
保护工业产权          保护文学艺术作品    1974成为UN专门机构     GATT+GATS+TRIPs三足鼎立
 │                       │                      │                   │
 ├─专利独立原则        ├─自动保护原则        ├─中国1980.3.3加入    ├─中国2001.12.11加入
 ├─国民待遇            ├─国民待遇            ├─管理24+国际条约     ├─强制执行机制
 └─外国优先权          └─最低保护标准        └─全球IP治理平台      └─贸易制裁挂钩
```

#### 4.2 巴黎公约（Paris Convention for the Protection of Industrial Property, 1883）

**【★重点】** 国际间第一个重要的知识产权国际条约。

**三大核心原则**：

| 原则 | 英文 | 内容 | 意义 |
|------|------|------|------|
| **专利独立原则** | Independence of Patents | 各国专利相互独立，一国对专利的授权/无效不影响他国 | 尊重国家主权，避免治外法权 |
| **国民待遇原则** | National Treatment | 各成员国必须给予其他成员国国民与其本国国民同等的 IP 保护 | 消除国别歧视，建立平等保护平台 |
| **外国优先权原则** | Right of Priority | 在一个成员国首次申请后，在一定期限内（发明12个月/外观商标6个月）向其他成员国申请时，以首次申请日作为申请日 | 防止在首次申请后被他人抢先在其他国家申请 |

**构成要件与法律后果**：
- 优先权行使条件：(1) 首次申请在成员国 (2) 在优先权期限内提出后续申请 (3) 前后申请人一致 (4) 前后申请保护的是同一发明创造
- 法律后果：后续申请视为在首次申请日提出——在优先权期间内的公开/使用不破坏新颖性

#### 4.3 伯尔尼公约（Berne Convention for the Protection of Literary and Artistic Works, 1886）

**【★重点】** 版权保护全球普及的基础性公约。

**核心原则**：
- **自动保护原则**（Automatic Protection）：版权自作品创作完成之时自动产生，无需登记/注册（与专利的"申请授权制"形成对比）
- **国民待遇原则**（National Treatment）
- **最低保护标准**（Minimum Standards of Protection）：各成员国必须至少达到公约规定的保护水平
- **独立保护原则**：保护水平不受作品来源国保护水平限制

**历史注脚 —— 美国与版权保护的反转**：
19 世纪的美国是移民社会，只给美国作者提供版权保护，大量盗版欧洲（尤其英国）作品。欧洲曾派代表团游说美国，未果。当美国从"文化输入国"转变为"文化输出国"（尤其是好莱坞+唱片工业崛起后），其立场发生 180 度逆转——从抵制版权保护变为全球最强硬的 IP 保护推动者。这一案例是课上老师反复强调的论点佐证：**知识产权制度有深刻的政治文化层面的投射，一国的 IP 立场不是抽象的正义观，而是由其在全球产业链中的实际位置决定的**。

#### 4.4 WIPO（World Intellectual Property Organization, 1967/1970）

**成立过程**：
- 1967 年：签订《建立世界知识产权组织公约》于斯德哥尔摩
- 1970 年：公约生效，巴黎联盟与伯尔尼联盟的国际局合并为 WIPO
- 1974 年：WIPO 成为联合国专门机构（specialized agency of the UN）
- 中国加入：1980 年 3 月 3 日

**功能定位**：
- 管理 24+ 个国际知识产权条约
- 全球 IP 治理的协商与协调平台
- 提供 IP 注册服务（如 PCT 体系、马德里体系、海牙体系等）

**WIPO 的局限性**：
- 协调性为主，缺乏强制执行机制——这是它后来被 WTO/TRIPs 体系"超越"的根本原因
- 发展中国家在 WIPO 框架下有一定话语权

#### 4.5 WTO/TRIPs（1994）—— "带牙齿的国际经济组织"

**【★重点】** TRIPs 被称为"带牙齿的"（with teeth），因其具有强制执行机制。

**TRIPs 的历史坐标**：
- 1994 年：乌拉圭回合谈判结束，TRIPs 作为 WTO 一揽子协定的一部分签署
- WTO 体系三足鼎立：GATT（货物贸易）+ GATS（服务贸易）+ **TRIPs（知识产权）**
- 中国加入 WTO：2001 年 12 月 11 日——自动受 TRIPs 约束

**TRIPs 的核心特征**：

| 特征 | 说明 |
|------|------|
| **最惠国待遇** (Most-Favoured-Nation Treatment, MFN) | 给予一个成员国的任何优惠/特权，必须立即无条件给予所有其他成员国——这是巴黎公约/伯尔尼公约没有的机制 |
| **最低保护标准** | 所有 WTO 成员必须达到 TRIPs 规定的 IP 保护最低水平 |
| **执行机制** (Enforcement) | 要求成员国建立民事、行政、刑事程序和边境措施以执行 IP 权利 |
| **争端解决机制** (Dispute Settlement) | 可将 IP 争端诉诸 WTO 争端解决机制，败诉方若不在合理期限内纠正，胜诉方可实施**贸易制裁**（trade sanctions） |
| **与贸易挂钩** | IP 保护水平与市场准入/贸易利益挂钩——发展中国家若不强化 IP 保护，可能面临出口受阻 |

**TRIPs 争议**：TRIPs 是否实质上是美国主导的、将美国标准全球化的工具？课程中反复强调的批判视角——知识产权的国际法是西方法律一元协同化（harmonization）的不平等产物。TRIPs 以及超出 TRIPs 的框架（TRIPs-plus），更加保护美国知识产权利益。

#### 4.6 PCT 专利合作条约（Patent Cooperation Treaty）

**【★重点】** PCT 体系经常被误解——它不是"国际专利授权"体系！

```
┌──────────────────────────────────────────────────────────────────┐
│                    PCT 两阶段流程                                 │
├──────────────────────────────────────────────────────────────────┤
│                                                                  │
```

**PCT 核心知识点**：

| 知识点 | 内容 |
|--------|------|
| 法律性质 | 专利领域进行**合作**（非授权）的国际性条约 |
| 不存在 PCT 专利 | 严格表述：PCT 是专利申请提交与检索的合作机制，专利授权权在各指定国专利局 |
| 两阶段划分 | **国际阶段**（提交+检索+公布+初步审查）→ **国家阶段**（各指定国按本国法独立审查授权） |
| 经济考量 | 进入国家超过 5 个时，PCT 途径比直接逐国申请费用更低 |
| 中国角色 | 1994.1.1 成为成员国，自动成为 PCT 受理局 + 国际检索单位（ISA）+ 国际初步审查单位（IPEA），中文是 PCT 申请语言之一 |

#### 4.7 欧洲专利条约（European Patent Convention, EPC）

| 知识点 | 内容 |
|--------|------|
| 机构 | 设立欧洲专利局（European Patent Office, EPO），已向全球开放 |
| 审查结果 | "专利树"（bundle of patents）——EPO 统一审查后，在各缔约国分别产生一国专利 |
| 权利行使 | **侵权保护权利专属于各国**——同一专利产品在不同国家的专利解释规则不同，可能导致对同一产品在不同国家作出不同判决 |
| 欧洲统一专利法院 | Unified Patent Court (UPC)——受英国脱欧（Brexit）和德国宪法挑战影响，仍在协商推进中 |

### （3）案例与适用

**PCT 情景应用题**：
中国申请人发明了一项技术，计划在中国、美国、日本、德国、韩国 5 个国家获得专利保护。方案 A：逐国分别申请；方案 B：通过 PCT 途径提交。

- **方案 A 成本**：5 份独立的翻译费 + 5 份独立的代理费 + 5 份独立的申请费，且必须在优先权日起 12 个月内完成全部申请
- **方案 B 成本**：1 份 PCT 申请（中文提交）+ 国际检索 + 国际公布后，在优先权日起 30 个月内（而非 12 个月）进入各国国家阶段——多出了 18 个月的决策和筹资时间

适用结论：**进入国超过 5 个，PCT 途径在经济上优于逐个申请。**

### （4）考试提示

- **简答题高频**："PCT 体系是否授予国际专利？"——标准答案：否。PCT 是合作条约，不是授权体系。专利授权权在各指定国。
- **论述题**：国际 IP 保护体系从巴黎公约到 TRIPs 的演变趋势——趋势是由"协调"走向"强制"，由"程序性合作"走向"实质性标准统一"
- **名词解释必须准确**：TRIPs 的全称（Agreement on Trade-Related Aspects of Intellectual Property Rights），巴黎公约的三大原则（独立/国民待遇/优先权）
- PCT 两阶段的流程要能画图说明
- TRIPs 与贸易制裁挂钩是区别于其他 IP 条约的核心特征

## 5. 全球化、知识产权与"五维空间"框架

### （1）核心概念与定义

**【★重点】** 本课程的研究视角是"全球视角"（global perspective）——这是邵科教授课程的标识性方法论框架。

> 什么是全球化（globalization）？全球化的根本特征是**西方规则的全球化**，而这种规则全球化最尖锐的矛盾就是知识产权。

这一论断是本课程全部论述的逻辑起点。

### （2）制度原理与体系

#### 5.1 "五维空间"分析框架

课程提出从五个维度理解知识产权全球治理：

```
                         五维空间 (Five-Dimensional Space)
                                  │
    ┌─────────┬──────────┬─────────┼──────────┬──────────┐
    │         │          │         │          │          │
  五维      四维       三维      二维       一维       零维
(西方思维) (框架行动论)(法律文本) (......)   (......)   (......)
    │         │          │
    │         │          └── 行动的结果：制定的法律
    │         │              （如TRIPs、各国IP法）
    │         │
    │         └── 西方规则全球化的实现方式
    │             框架行动论（绝非科技社会发展论）
    │             认知的行动呈现
    │
    └── 西方的思维、认知
        最深层次的文化与认知范式
        IP制度的底层逻辑源于西方思维
```

**框架解析**：

| 维度 | 含义 | 在本课程中的体现 |
|------|------|-----------------|
| **五维** — 西方的思维与认知 | 最深层的文化/认知范式。IP 的底层逻辑（个体权利、排他性财产权、创新激励论）本质上是西方思维的产物 | 课程强调 de-Eurocentric（去欧洲中心化），要求从非西方视角审视 IP |
| **四维** — 框架行动论 | 西方规则全球化的实现方式。不是科技/社会自然发展的结果，而是有目的、有框架的行动输出 | TRIPs 是"框架行动"的产物——美国通过贸易谈判将自身 IP 标准强制全球化 |
| **三维** — 法律文本 | 行动的结果——制定的法律条文 | 巴黎公约、伯尔尼公约、TRIPs、各国 IP 法 |

**关键区分**：邵科教授强调，IP 的全球化绝非"科技社会发展论"——即不是技术进步自然而然推动各国制定 IP 法的过程。而是**框架行动论**——西方国家（尤其美国）通过制度性框架（WTO/TRIPs、双边贸易协定中的 TRIPs-plus 条款、贸易制裁威胁等）有计划地将其 IP 标准推向全球的过程。

#### 5.2 国际法 vs 全球法

**【★重点】** 课程区分了两个关键概念：

| 概念 | 英文 | 含义 | 评价 |
|------|------|------|------|
| **知识产权的国际法** | International IP Law | 美国主导的，TRIPs 模式以及超出 TRIPs 的框架（TRIPs-plus），更加保护美国 IP 利益的规则体系 | 西方法律一元协同化（harmonization）的不平等产物 |
| **知识产权的全球法** | Global IP Law | 去西方中心化（de-Eurocentric）的全新概念——强调多元文化主体的平等参与和共同建构 | 课程努力的方向 |

**核心论点**：现有的"国际知识产权法"不是真正意义上的"国际"法（平等参与、多元协商），而是**西方规则的不对等全球化**。真正的"全球知识产权法"应当：
- 去西方中心化（de-Eurocentric）
- 文化包容（culturally inclusive）
- 跨学科（cross-disciplinary）
- 呈现"中国本位·全球经验"

#### 5.3 美国知识产权产业的群体策略（Business Stakeholders and Strategic Partners）

课程笔记指出：美国知识产权产业的群体策略面向。美国的 IP 政策不仅是政府行为，更是**产业界深度参与**的结果。典型机制：
- 美国贸易代表办公室（USTR）的"特别 301 报告"——单方面将其他国家列入"重点观察名单"，施压其强化 IP 保护
- 制药、娱乐、软件等 IP 密集型产业通过游说直接影响美国贸易谈判立场
- 双边/区域贸易协定中嵌入 TRIPs-plus 条款（比 TRIPs 更严格的保护标准）

### （3）案例与适用

**TRIPs-plus 现象**：
美国在与贸易伙伴的双边/区域 FTA 谈判中，经常要求对方接受超出 TRIPs 最低标准的 IP 保护条款——例如更长的专利保护期、数据独占权（data exclusivity）、专利链接制度（patent linkage）等。此类条款被称为 TRIPs-plus，实质上是利用市场准入作为杠杆，推行高于多边框架的保护标准。

这一现象的法学意义：TRIPs 作为多边框架的"地板"（最低标准）被美国用作"天花板"的跳板——先以 TRIPs 确定全球基线，再通过双边渠道不断提升基线。

### （4）考试提示

- **论述题**："全球化的根本特征是西方规则的全球化，而这种规则全球化最尖锐的矛盾就是知识产权"——这句话要能展开论述（为什么"最尖锐"？因为 IP 涉及核心经济利益+文化话语权）
- **名词辨析**：International IP Law vs Global IP Law——两个概念的含义和价值取向
- **框架行动论 vs 科技社会发展论**——老师的重要立场，论述题中要体现对这一区分的理解
- TRIPs-plus 是理解当代 IP 国际保护的重要概念，可能出现在论述题中作为批判素材

### （3）全球视角的方法论含义

课程强调"全球视角"不是简单的"比较法"——不是仅比对不同国家法律的差异。全球视角的核心是：

1. **跨学科**（cross-disciplinary）：融合法学、经济学、政治学、历史学等多学科方法
2. **文化包容**（culturally inclusive）：突破西方中心主义的 IP 叙事，容纳非西方文化传统对"知识""创造""财产"的不同理解
3. **去欧洲中心化**（de-Eurocentric）：批判性地审视 IP 规则全球化的过程，揭示其并非"普适价值"的自然扩散，而是特定文明形态的制度输出

## 📋 本章考点速记

| 序号 | 考点 | 一句话理解 | 答题要点 |
|------|------|-----------|---------|
| 1 | **IP 定义困境** | IP 没有统一定义，通过举例来描述；IP Law 调整人类对智力/创造性劳动成果的创造、使用和利用 | 统一定义缺失 + 举例式方法 + 无形性 + 三层困境原因 |
| 2 | **IP 无形性 (Intangible)** | IP 本质无形，参照有形财产规则构建，容易造成过度垄断 | 无形 vs 有形的四点对比（排他性/边界/保护方式/期限）+ 类比规制的利弊 |
| 3 | **自然权利 vs 国家授予** | IP 是自然权利还是国家法定权利？课程立场倾向于国家授予说 | 两种学说对比 + 自然权利说的困难（为何有期限？为何需法律确认？） |
| 4 | **Monopoly vs Exclusive Rights** | 西方称为 monopoly（负面），中性表述为 exclusive rights（核心词） | 表述差异 + 政策含义 + 话语权争夺 |
| 5 | **公有领域 (Public Domain)** | 公有领域是 IP 保护的外部边界——发现、公式、自然规律不受保护 | 排除原则 + 期限限制度 + 公有领域范围是否过小的争议 + 开源运动挑战 |
| 6 | **WIPO 公约 8 大类** | 最广泛的 IP 种类列举（1967），含科学发现（争议）+ 兜底条款 | 8 类逐项列举 + 科学发现的争议地位 |
| 7 | **TRIPs 协定 7 大类** | 有美国特色的 IP 分类（1994），含地理标志和集成电路布图设计 | 7 类逐项列举 + 与 WIPO 的 4 点差异 |
| 8 | **民法典第123条** | 中国 IP 保护的民事基本法规范：8 类客体 + "专有权利"表述 + 兜底→数据 | 结构五要素分析（主体/来源/性质/客体/兜底）+ 植物新品种的中国特色 |
| 9 | **巴黎公约三大原则** | 专利独立 + 国民待遇 + 外国优先权（1883，第一个重要 IPR 条约） | 三原则含义 + 优先权构成要件与法律后果 |
| 10 | **伯尔尼公约** | 自动保护原则——版权自创作完成时自动产生，无需登记 | 自动保护 vs 专利申请授权制的对比 + 美国版权反转的历史案例 |
| 11 | **WIPO** | 巴黎+伯尔尼联盟合并（1967/1970），1974 成为 UN 专门机构 | 成立过程 + 功能定位 + 缺乏强制执行力的局限性 |
| 12 | **WTO/TRIPs** | "带牙齿的"IP 协定——最惠国待遇 + 争端解决 + 贸易制裁挂钩 | 五特征 + 三足鼎立（GATT+GATS+TRIPs）+ TRIPs 的批判视角 |
| 13 | **PCT 两阶段** | 不是"国际专利授权"！国际阶段（提交+检索+公布+初审）→ 国家阶段（各局独立授权） | 不存在 PCT 专利 + 两阶段流程图 + 超 5 国经济优势 + 中国三重角色 |
| 14 | **欧洲专利条约** | EPO 统一审查→"专利树"→侵权保护权专属各国 | 统一审查 + 各国独立解释 + UPC 受 Brexit 和德国宪法挑战影响 |
| 15 | **五维空间框架** | 五维（西方思维）→四维（框架行动论）→三维（法律文本） | 三个层次的含义 + 框架行动论 vs 科技社会发展论的区分 |
| 16 | **国际法 vs 全球法** | 国际法 = 西方一元协同的不平等产物；全球法 = 去西方中心化的新构想 | 两概念对比 + 课程价值取向 |
| 17 | **TRIPs-plus** | 超出 TRIPs 最低保护标准的更高保护条款（双边/区域 FTA 中的美国策略） | 概念定义 + 产生机制 + 与多边框架的关系 |
| 18 | **IP 保护期限三档** | 专利 15-20 年 / 著作权 70-170 年 / 商标无限期 | 三档对比 + 各自的政策理由 |
| 19 | **科学发现 vs 发明** | 发现不保护（公有领域），发明可受专利保护 | 区分标准 + E=mc² 思想实验 + 中国无独立科学发现权 |
| 20 | **美国版权反转** | 从盗版国到全球最大 IP 保护推动者——IP 立场由产业链位置决定 | 历史事实 + 法学意义（IP 有政治文化层面的投射） |
